# Servo-Motor
This code uses a Nano Supermini ESP32 to control the angle of a 0 to 180 degree servo motor. The program prompts you to enter an angle of 0 to 165 degrees. It will not go to 180 degrees due to the physical limitations of the motor. If you enter an angle less than 0 or greater than 165 it will prompt you that it is an invalid input and will then print the original prompt.
